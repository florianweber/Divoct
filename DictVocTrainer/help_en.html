Context help
You can get context-relevant help by tapping with two fingers on the screen anywhere.
Gestures in search tab
Sometimes you might want to search several words quickly after each other, for example if you read a book. In the "Search" tab, you can move around quickly be using swipe gestures to the left and right. If - for example - you are looking at the details / translation screen, you can get back to search by swiping right. Swipe left to get to the folder assignments.
Shake for a new search
If you shake your device anywhere, you will get back to the search. A good place to use this technique is the details / translation screen, because this is the fastest way to enter a new search term.
Tap to open translation detail
In the details view of a word you see one or more translations. If you tap one of them, this word will now be shown in the details view.
Limit correct answers for training
A word in one language often has many different meanings in another language. This can be quite confusing in trainings. Therefore you can limit correct answers for your trainings. Tap the edit icon in the details view of a word and select the answers you want to be shown in trainings. By default, all answers will be selected. For faster access to the edit menu, just double tap on the screen in details view.
